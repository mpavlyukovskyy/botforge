# BotForge Competitive Research

## Market Gap

No existing project combines YAML-driven config + multi-platform deployment + fleet lifecycle dashboard + inter-bot communication.

## Unique Positioning

BotForge is the **operational layer** for AI agents — "Portainer for AI bots". Not another framework; the platform to run, monitor, and manage agents at scale.
